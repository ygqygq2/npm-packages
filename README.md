# npm-packages

ygqygq2's npm packages.

## What's inside?

This Turborepo includes the following packages/apps:

### Apps and Packages

- `@ygqygq2/eslint-config`: eslint configurations
- `@ygqygq2/react-datepicker-mui`: a date picker component for React using Material-UI
- `@ygqygq2/stylelint-config`: stylelint configurations
- `@ygqygq2/typescript-config`: `tsconfig.json`s
- `@ygqygq2/utils`: common utils
- `@ygqygq2/vscode-log`: vscode extension channel logger

## Maintenance

- When modifying code, you don't need to manually update the `version` field in `package.json`.
- After pushing to GitHub, GitHub Actions will automatically publish new packages with updated versions.
